# Release Notes - v3.3.1

## Fixes
- **Windows self-update handoff**: Fixed a PowerShell `Wait-Process` warning that could appear when the original `onesource update` process had already exited before the background replacement script started.
- **Update only when needed**: `onesource update` now checks the latest GitHub release version before downloading. If the installed version is already current, it prints `Already up to date.` and exits without replacing the executable.

## Notes
- This is a patch release for the self-update feature introduced in v3.3.0.
