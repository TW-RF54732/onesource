# Release Notes - v3.0.5

## Improvements
- **Increased Resilience**: The `--save` command now handles incompatible or corrupted configuration files by overwriting them instead of failing, improving user experience during migrations.
- **Better Test Isolation**: Improved the integration test suite to ensure clean state between test steps.

## Bug Fixes
- Fixed a CI failure where legacy configuration compatibility tests were blocking subsequent profile management tests.
- Updated test expectations to align with the new application-level error handling (using `Result` instead of warnings).
