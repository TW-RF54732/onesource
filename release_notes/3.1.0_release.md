# Release Notes - v3.1.0

## New Features
- **Profile Descriptions**: Profiles can now have descriptions to help identify their purpose.
- **Enhanced Profile Management**:
    - Added `--desc <description>` flag to the main command for setting descriptions when saving a profile.
    - Added `profile desc <description> -p <profile>` subcommand to update descriptions of existing profiles.
    - Automatic preservation of existing descriptions when updating other profile settings.

## Improvements
- **CI/CD Testing**: Added comprehensive integration tests for profile description management to ensure reliability.
- **CLI Consistency**: Updated internal versioning to 3.1.0 across the project.
