# Release Notes - v3.5.1

## Highlights

- Published `onesource` as an installable crate with complete crates.io metadata.
- Limited the crate package to the source, lockfile, README, and license so release media and repository automation are not shipped to Cargo users.
- Added `cargo install onesource` to the English and Traditional Chinese installation guides.
- Added a native Apple Silicon (`aarch64-apple-darwin`) release binary while retaining Intel macOS support.
- Updated the Unix installer to select the correct macOS binary for Intel or Apple Silicon.
- Made the CLI version derive from `Cargo.toml` to prevent release version drift.

## Verification

- Ran `cargo clippy -- -D warnings`.
- Ran `cargo test` (27 tests).
- Ran `cargo build` and all 17 CLI integration scenarios.
- Ran `cargo publish --dry-run` and inspected the 14-file package (31.8 KiB compressed).
