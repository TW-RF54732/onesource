# Release Notes - v3.0.1

## Improvements
- **Robust Error Handling**: Integrated `anyhow` for application-level error management, providing more context-rich error messages.
- **Idiomatic Code Quality**: Refactored core logic in `src/configs.rs` and `src/main.rs` to follow Rust best practices, improving maintainability and performance.
- **Improved CLI Experience**: Replaced manual panics and silent exits with clear, actionable error reporting for configuration and file system operations.
- **Cleanup**: Renamed internal functions to better reflect their intent and reduced unnecessary data cloning.

## Bug Fixes
- Fixed potential issues with configuration merging and profile loading by ensuring consistent error propagation.
