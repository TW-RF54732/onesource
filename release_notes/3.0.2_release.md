# Release Notes - v3.0.2

## Improvements
- **Standardized Code Formatting**: Applied `rustfmt` compliant styling across the entire codebase to ensure CI consistency.
- **Robust Error Handling**: Integrated `anyhow` for application-level error management with context-rich error messages.
- **Idiomatic Rust Refactoring**: Core logic in `src/configs.rs` and `src/main.rs` has been refactored for better maintainability.
- **Automated CI/CD**: Updated GitHub Actions to automatically extract release descriptions from the `release_notes/` directory.

## Bug Fixes
- Fixed CI build failures caused by non-standard formatting in previous releases.
- Resolved minor logical inconsistencies in configuration merging.
